# Application Icons

Please place the following icon files in this directory:

## System Tray
- `tray-icon.ico` - Windows
- `tray-icon.png` - Linux
- `iconTemplate.png` - macOS (Template image, usually black and transparent)

## Application Icons (for Builder)
- `icon.ico` - Windows
- `icon.icns` - macOS
- `icon.png` - Linux (512x512)
